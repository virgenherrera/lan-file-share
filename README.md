## Description

basic [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

1. install dependencies.

```bash
$ npm install
```

2. Create `.env` file.

```bash
$ cp ./env.example ./env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
